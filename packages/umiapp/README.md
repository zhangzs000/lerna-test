# umi project

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

### 开始项目
- 初始化
```
lerna init
yarn create @umijs/umi-app // 脚手架生成项目
// 初始 使用 yarn add umi，不知为什么node_modules都安装到了 lerna勒纳 顶层目录上了
```
- script
```
lerna勒纳 配置script ,必须给对应的项目package.json 添加 name字段。
```
